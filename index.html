My experience attending a Coding Bootcamp

While attending the bootcamp at Coding Dojo they taught us 3 stacks for full CRUD web development. This was a Full-time program requiring 8-10 hours a day on average. A typical day would start with an algorithm session where we would break into small groups to tackle a new coding challenge each day. The challenges they assigned always required us to think outside the box and collaborate with the other cohorts in our group. This was good for team building as well as thinking how to break the problem down into manageable chunks. We started with the stack of Python, Flask, and MySQL. We spent a month on each stack, doing the course work assigned. Towards the end of each stack we were required to come up with a project idea using what we learned in class as well as some tool/framework/library we found outside of class. For my Python project I learned how to use PyGame and created a mario like side scroller called Devio (more info on projects page).
