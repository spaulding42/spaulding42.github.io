#Vanlife App

Over the past few years, I've been interested in the Vanlife movement. I built an empty cargo van into a livable camper van and built an app to showcase how to build a van and have a forum where other Vanlifers can ask questions and interact with one another.

This app utilizes the following technologies:

  • C#/.NET
  • Bootstrap and CSS
  • A robust MySQL Database consisting of Users, Posts, and Comments
  • MySQL Workbench
